# PBM Project - Weather App

## Deskripsi
Aplikasi ini dibuat sebagai tugas **Pemrograman Berbasis Mobile** menggunakan **Flutter**.  
Aplikasi ini berfungsi untuk **menampilkan cuaca saat ini dan ramalan cuaca 5 hari ke depan** di lokasi yang dipilih pengguna.

## Fitur
- Menampilkan cuaca saat ini: suhu, kondisi cuaca, kelembapan, dan kecepatan angin.  
- Ramalan cuaca 5 hari ke depan dengan informasi suhu minimum & maksimum dan kondisi cuaca.   
- Tampilan UI sederhana dan interaktif.
